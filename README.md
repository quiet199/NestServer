用于构建高效且可伸缩的服务端应用程序的渐进式 Node.js 框架。

完美支持 Typescript
面向 AOP 编程
支持 Typeorm
高并发，异步非阻塞 IO
Node.js 版的 spring
构建微服务应用
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
